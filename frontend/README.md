## 💡 ПРИНЦИПЫ АДАПТИВНОСТИ:

### 1. Используйте `clamp()`:
```typescript
fontSize: 'clamp(14px, 2vw, 18px)'
// минимум: 14px (мобильный)
// предпочтительно: 2vw (относительно экрана)
// максимум: 18px (десктоп)
```

### 2. Всегда добавляйте:
```typescript
boxSizing: 'border-box'  // Правильный расчёт размеров
flexWrap: 'wrap'         // Перенос элементов
minWidth: 0              // Для flex с текстом
aspectRatio: '1 / 1'     // Для круглых элементов
```

### 3. Grid с авто-колонками:
```typescript
gridTemplateColumns: 'repeat(auto-fit, minmax(min(280px, 100%), 1fr))'
// Автоматически подстраивает количество колонок под ширину
```

```typescript

minWidth: 0
```

### Круги стали овалами:
```typescript
// Добавьте:
aspectRatio: '1 / 1'
```

## 📞 ПОДДЕРЖКА:

Если что-то не работает:

2. Проверьте импорт: `import { responsive } from '../styles/responsive'`
3. Проверьте Chrome DevTools на разных разрешениях (F12 → Toggle Device Toolbar)
